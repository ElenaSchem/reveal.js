SVG

Scalable Vector Graphics

General information

SVG

is a language for describing two-dimensional graphics in XML*

*according to the book "Pocket Guide to Writing SVG" by Joni Trythall

Simple example

History

  • Came about in 1999
  • SVG is supported by all major browsers
  • Loading SVG can be slow
  • DOM interface available for it
  • Not requiring third-party extensions

Use cases


									
							

								.logo {
									background-image: url(logo.svg);
								}

Fills and Strokes

Fill

paints the interior of a specific graphical element, paints open paths

  • fill-rule (nonzero, evenodd, inherit)
  • fill-opacity

Stroke

defines the “border” paint of particular shapes and paths

  • stroke-width
  • stroke-linecap
  • stroke-linejoin
  • stroke-miterlimit
  • stroke-dasharray
  • stroke-dashoffset
  • stroke-opacity
